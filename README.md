# Image Diff Review

*Image diff review* is a tool for reporting image differences. It is intended for use in snapshot testing.

<img src='docs/screenshot.png' width='100%'>

## Usage

```commandline
$ image_diff_review <left/image_dir> <right/image_dir> report
```
